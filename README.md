This project is a wireless blimp control and monitoring system built using an ESP32 camera module and a Python-based ground control station. The system allows an operator to remotely control the blimp's movement while receiving a live video feed from the onboard camera. Three motors are controlled independently: two propulsion motors provide forward movement and steering through differential thrust, while a third motor controls vertical ascent and descent. The Python application provides an intuitive graphical interface with real-time video display, thrust control, yaw adjustment, vertical movement control, and an emergency stop function.

The system is achieved through Wi-Fi communication between the ESP32 and the desktop application. The ESP32 hosts an HTTP server that captures grayscale images from the camera and responds to motor control commands sent by the Python client. The ground station is developed using Python with Tkinter for the GUI, OpenCV and NumPy for image processing, Pillow for image display, and Requests for network communication. By combining embedded hardware control, wireless networking, computer vision, and a user-friendly interface, the project demonstrates a complete remotely operated aerial vehicle control system.
